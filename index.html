Class Projects

Hello World.

After spending almost 15 years as a paralegal with heavy emphasis in electronically stored informaion (ESI) using e-discovery data managment technologies, I am ransitioning into web development while working as an e-DAT analyst for a large international law firm processing ESI. I consider this my stepping stone on my path to becoming a front-end web developer, with goals of working as a full-stack developer, and eventually into Project Management.

Using the The Cocktail DB API, I created a fun little page where you can type in a drink name and it will show you the ingredients and how to make it. If you can't think of one, you have an option to have the API give you a random drink.

Using APIs

With all the data running around in the world, why not use an API to make it easier to create fun little interactive web apps? In WATS4000 we learned how to use free APIs to create an multi-page webapp using Datamuse.

A Day at the Beach

For my WATS3020 JavaScript final project, I created an interactive page that highlights some of the things one might see on a trip to the beach here in the Pacific Northwest. A click on the image will provide the taxonomy and a little fact about the animal or plant seen.
